# Micro apps Angular + Vue

### Installation
```
npm run installs
```

### Dev
Angular shell run:
```
npm run serve:shell
```
MY MICRO APP run:
```
npm run dev:my-wc
```
CLIENT-A run:
```
npm run dev:client-a
```

### Build micro apps
MY MICRO APP run:
```
npm run build:my-wc
```
CLIENT-A run:
```
npm run build:client-a
```

### Serve micro apps dist together
```
npm run serve:my-wc
```
```
npm run serve:client-a
```
